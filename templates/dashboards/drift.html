About Drift Monitoring

Inter-Story Drift Analysis for Structural Health Monitoring

System Overview

This module is designed to monitor inter-story drift behavior of structures using vibration-based analysis. It estimates displacement from acceleration signals captured by seismic sensors and evaluates structural performance relative to baseline models derived from ETABS.

Methodology

  • • Acceleration signals are collected from RS4D seismic sensors
  • • Signals are processed to compute RMS acceleration
  • • Displacement is derived using frequency-domain relationships
  • • Inter-story drift ratios are calculated per floor
  • • Results are compared against ETABS baseline values

Key Parameters

Drift Ratio

Measure of relative displacement between floors.

Baseline Drift

Reference values obtained from ETABS structural model.

Acceleration

Measured ground motion converted to structural response.

Drift Difference

Deviation between measured drift and baseline.

System Information

This module is part of an integrated Structural Health Monitoring (SHM) system combining frequency domain analysis (NFD) and drift-based evaluation for real-time structural assessment.
